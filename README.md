# NetworkUp

## Description
> Level up your networking skills by completing tasks and challenges in this gamified, event-based mobile application.

## Deployed Site
* https://boiling-mesa-67164.herokuapp.com/

## Requests

### GET
* /badges/:id
* /events
* /events/byeventid/:id
* /events/:userid
* /events/goals/:id
* /events/retros/:id
* /events/retros/byuser/:id
* /challenges
* /challenges/:id
* /account_challenges/:id

### POST
* /events

### PUT
* /events/goals/:id
* /events/checkout/:id
* /events/retros/:id
